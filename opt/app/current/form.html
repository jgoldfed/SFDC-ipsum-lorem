Salesforce - Generate Test Data

The purpose of this form is to generate test data for any Salesforce object. Enter your credentials and the test data parameters and click Submit to generate random text in your browser.

  • Enter the Salesforce URL (e.g.,https://site--dev1.cs32.my.salesforce.com)

  • Enter the Salesforce User Name

  • Enter the Salesforce Password

  • Test Data Parameters

  • Enter the Object Name

  • Enter the Number of Rows to Generate
